# Geepers Skills Collection

A consolidated set of skills for the [geepers](https://github.com/lukeslp/geepers) ecosystem, organized into 7 focused groups.

## Skills

| Skill | Description |
|-------|-------------|
| [orchestration](skills/orchestration/) | Multi-agent coordination, Dream Cascade/Swarm, service management |
| [planning](skills/planning/) | Task prioritization, building, scouting, quality auditing, testing, validation |
| [research](skills/research/) | Data fetching (17+ APIs), corpus linguistics, etymology, LLM access, parallel swarm |
| [visualization](skills/visualization/) | D3.js, Chart.js, plotly, color design, UX analysis, data storytelling |
| [devtools](skills/devtools/) | Git hygiene, deployment, session start/end rituals |
| [content](skills/content/) | Documentation generation, technical writing, product management |
| [humanize](skills/humanize/) | Remove AI writing indicators from documentation and prose |
